# Archived launcher scripts

The scripts in this folder are historical references only.

Use `scripts/run_openrouter_tournament.py` for active OpenRouter tournament runs.

If you are looking for a retired one-off script, its functionality is usually covered by one of these canonical modes:

- `--schedule-mode single-round-robin`
- `--schedule-mode double-round-robin`
- `--schedule-mode limited-coverage`
- `--model-preset top4`

Retired files are kept here so older notes and transcripts can still be understood, but they should not be used as the active entrypoint.
